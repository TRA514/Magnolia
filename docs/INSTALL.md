# Installing Magnolia

Pick your platform — each guide is self-contained, so you can hand a teammate the single file for
their OS:

- **macOS** → [`INSTALL-macos.md`](./INSTALL-macos.md)
- **Windows** → [`INSTALL-windows.md`](./INSTALL-windows.md)

Both walk through the same shape: install prerequisites, land the repo *where your Claude Code
already works* (so it inherits your connectors + skills), restart Claude Code, then `onboard me`.
